## Paul's website house style (use when building or designing sites)
- **One self-contained `index.html`**: Tailwind via CDN + inline config, **GSAP 3.12 from
  cdnjs** (ScrollTrigger + MotionPath). No build step. Always a fixed inline-SVG
  **feTurbulence film-grain** overlay (~0.05 opacity, mix-blend overlay).
- **Palettes:** (a) near-black `#080808` + metallic-gold gradient `#D4AF37 → #8a6531`; or
  (b) warm cream `#F5EFE6` + forest/sage + champagne. Adapt to the brief.
- **Type:** elegant serif display (Cormorant Garamond / Cinzel / Playfair) + clean sans
  (Manrope / Inter / Jost). Heavy display (Anton/Bebas) for loud briefs.
- **Signature GSAP:** scripted hero timeline (curtain/door/clip-path reveal), ScrollTrigger
  pin + scrub, one pinned horizontal-scroll section, MotionPath dot on an SVG route,
  expanding hairline dividers, `mix-blend-difference` nav, `-webkit-text-stroke` outline
  headings, marquee strip, count-up stats, magnetic buttons.
- **Motion grammar:** easings `power2/3/4.out` + `expo.inOut`, tight staggers (0.02–0.08s),
  scrub ≈ 1. **Restraint over flash** — every animation has a reason; keep it fast.
- For his very best sites he writes a full creative brief and runs it through **Gemini 3 Pro**.
